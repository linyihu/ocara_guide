模板引擎


1、模板引擎 
      一般地，模板文件直接使用原生的HTML嵌套PHP代码。自从出现了模板引擎，这项工作变得更加方便易用。
      本框架已经集成了smarty模板，只要配置一下即可。而且还提供了模板引擎扩展。

2、配置smarty模板
（1）开启smarty

/*
 * 模板配置
 */
'TEMPLATE' => array(
    'file_type'		 => 'php', //模板文件名
    'engine'		 => 'Ocara\Service\Smarty', //模板引擎（如果使用默认的Smarty模板引擎，填Ocara\Service\Smarty）
    'default_style'  => 'defaults', //默认模板风格目录
    'default_layout' => 'layout', //默认的layout名称
),

      将其中的engine设置为smarty就可以开启smarty，将file_type设置为模板文件要使用的文件扩展名，如html或者htm等等。
  
（2）配置smarty

/*
 * Smarty模板配置
 * 可自由选择是否使用smarty模板
 */
'SMARTY' => array(
    'use_cache'	 => 0, //是否使用缓存
    'left_sign'	 => '<{', //模板中左标记
    'right_sign' => '}>', //模板中右标记
),


3、使用其他的模板引擎
      如果您想要的模板引擎Ocara没有提供支持，您可以很轻松的实现。点击进入：模板引擎扩展。
