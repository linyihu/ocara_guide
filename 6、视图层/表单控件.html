表单控件的生成（Html）


    表单控件类是Ocara\Core\Html，表单类Ocara\Core\Form已经将其集成为插件。
    以下是该组件生成表单控件的方法函数：

（1）input(type, name[, attributes = false, checked = false])
      生成Input标签元素。
      参数说明：
名称 描述
type 类型。INPUT元素类型，即type属性值，如“button”，“text”等。
name 必需。名称
attributes 可选。同上。

以下实例显示：    

//带上属性placeholder
echo ocService()->html->input(
	'text',
	'username',
	array(
		'placeholder' => '请输入用户名'
	)
);

//用户备注默认值为"暂无备注"
echo ocService()->html->input('text', 'user_note', '暂无备注');


（1）text(name[, attributes])
      生成文本框。
      参数说明：
名称 描述
name 必需。名称
attributes 可选。如果是关联数组，用来指定标签属性，键名为属性名，键值为属性值。如果是字符串或者数字等，则指定的是标签的value属性值。 

（2）password(name[, attributes])
      生成密码框。
      参数说明：
名称 描述
name 必需。同上。
attributes 可选。同上。 

（2）button(name[, attributes])
      生成按钮。
      参数说明：
名称 描述
name 必需。同上。
attributes 可选。同上。 

（3）file(name[, attributes])
      生成上传文件控件。
      参数说明：
名称 描述
name 必需。同上。
attributes 可选。同上。 

（4）hidden(name[, attributes])
      生成隐藏域。
      参数说明：
名称 描述
name 必需。同上。
attributes 可选。同上。 

（5）textarea(name[, attributes])
      生成文本域。
      参数说明：
名称 描述
name 必需。同上。
attributes 可选。同上。 
（6）radio(name, desc[, attributes, checked]) 
      生成单选框。
      参数说明：
名称 描述
name 必需。同上。
desc 必需。复选框右边的文字。
attributes 可选。同上。
checked 可选。checked属性值，指定是否选中单选框。

以下实例显示成： 男 

echo ocService()->html->radio(
	'sex_man',
	'男',
	array(
	   'value' => 0
	),
	true //初始化时选中单选框
);

（7）radioMulti(name, radios[, checked, attributes])
       生成批量单选框。单选框之间以空格分隔。
名称 描述
name 必需。同上。
radios 必需。单选框数组。键名是单选项的value，键值是单选框右边的文字描述。
checked 可选。选中值，字符串或数组。
attributes 可选。同上。

以下实例打印成数组： Array ( [0] => 男 [1] => 女 [2] => 未知 ) 

$radioList = ocService()->html->radioMulti(
	'sex',
	array(
	   	'0' => '男',
	   	'1' => '女',
		'2' => '未知',
	),
	'1', //默认选中女
	array(
	   'value' => 1
	)
);

print_r($radioList);

（8）checkbox(name, desc[, attributes, checked])




	

		名称
		描述
	

	

		name
		必需。同上。
	

		desc
		必需。复选框右边的文字。
	

	

		attributes
		可选。同上。
	

		

		checked
		可选。同上radio()。
	





（9）checkboxMulti(name, checkboxes[, checked, attributes])
       生成批量复选框。复选框之间以空格分隔。
	   返回复选框HTML代码数组。




	

		名称
		描述
	

	

		name
		必需。同上。
	

		checkboxes
		必需。复选框数组。键名是复选项的value，键值是复选框右边的文字描述。
	

	

		checked
		可选。选中的值数组。
	

		

		attributes
		可选。同上。
	






以下实例打印成数组：
Array
(
    [0] => 百度
    [1] => 谷歌
    [2] => 必应
)




$checkboxList = ocService()->html->checkboxMulti(
	'search_type',
	array(
	   	'1' => '百度',
	   	'2' => '谷歌',
		'3' => '必应',
	),
	array('1', '3') //选中百度和必应
);

print_r($checkboxList);



（10）select(name, options[, attributes, nullText]) 
      生成下拉框




	

		名称
		描述
	

	

		name
		必需。同上。
	

		options
		必需。option选项列表数组，键名选项值即value属性值，键值是显示文本
		  
	

	

		attributes
		可选。选中值或属性数组。如果是字符串或数字，表示默认选中的选项值；如果是属性数组，添加键名为value元素即可。 
	

		

		nullText
		可选。下拉框一般有一个空选择的显示文本，如“请选择”，其值为空，表示未选中任何选项。
	






	 以下实例显示为：
	




$options = array(
	'0' => '男',
	'1' => '女',
	'2' => '未知',
);

$attributes = array(
   'value' => '1', //选中的选项
);

echo ocService()->html->select('sex', $options, $attributes, '请选择...');



（11）selectGroup(name, options[, attributes, nullText])
      生成分组下拉框。




	

		名称
		描述
	

	

		name
		必需。同上。
	

		options
		必需。同上。
	

	

		attributes
		可选。同上。 
	

		

		nullText
		可选。同上。
	






	 以下实例显示为：
	




$options = array(
	array(
		'optgroup' => '电商', //分组名称
		'options' => array(
			'taobao' => '淘宝',
			'jingdong' => '京东',
		),
	),
	array(
		'optgroup' => '搜索', //分组名称
		'options' => array(
			'baidu' => '百度',
			'google' => '谷歌',
		),
	),
);

$attributes = array(
   'value' => 'baidu', //选中的选项
);

echo ocService()->html->selectGroup('website_groups', $options, $attributes, '请选择...');



（12）options(options[, value, nullText])
      仅生成option选项列表。如果不加select框，只生成option列表框，可以使用该方法。
      参数说明。




	

		名称
		描述
	

    

		options
		必需。同上。每个选项组的第一个选项的选项值用数组，并且添加optgroup属性即可。
	

	

		attributes
		可选。同上。 
	

		

		nullText
		可选。同上。
	






      以下实例返回字符串：<option value="">请选择...</option><option value="0">男</option><option value="1" selected="selected">女</option><option value="2">未知</option>



$options = array(
	'0' => '男',
	'1' => '女',
	'2' => '未知',
);

$optionsHtml = ocService()->html->options($options, 1, '请选择...');
