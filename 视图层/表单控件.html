表单控件的生成（Html）


    表单控件类是Html，表单类Form已经将其集成为插件。
    以下是该组件生成表单控件的方法函数：

（1）input(type, name[, attributes = false, checked = false])
      生成Input标签元素。
      参数说明：
名称 描述
type 类型。INPUT元素类型，即type属性值，如“button”，“text”等。
name 必需。要保存的xml文件路径
attributes 可选。如果是关联数组，用来指定标签属性，键名为属性名，键值为属性值。如果是字符串或者数字等，则指定的是标签的value属性值。 

（1）text(name[, attributes])
      生成文本框。
      参数说明：
名称 描述
name 必需。要保存的xml文件路径
attributes 可选。如果是关联数组，用来指定标签属性，键名为属性名，键值为属性值。如果是字符串或者数字等，则指定的是标签的value属性值。 

（2）password(name[, attributes])
      生成密码框。
      参数说明：
名称 描述
name 必需。同上。
attributes 可选。同上。 

（2）button(name[, attributes])
      生成按钮。
      参数说明：
名称 描述
name 必需。同上。
attributes 可选。同上。 

（3）file(name[, attributes])
      生成上传文件控件。
      参数说明：
名称 描述
name 必需。同上。
attributes 可选。同上。 

（4）hidden(name[, attributes])
      生成隐藏域。
      参数说明：
名称 描述
name 必需。同上。
attributes 可选。同上。 

（5）textarea(name[, attributes])
      生成文本域。
      参数说明：
名称 描述
name 必需。同上。
attributes 可选。同上。 
（6）radio(name, desc[, attributes, checked]) 
      生成单选框。
      参数说明：
名称 描述
name 必需。同上。
desc 必需。复选框右边的文字。
attributes 可选。同上。
checked 可选。checked属性值，指定是否选中单选框。 
（7）radioMulti(name, radios[, checked, attributes])
       生成批量单选框。单选框之间以空格分隔。
名称 描述
name 必需。同上。
radios 必需。单选框数组。键名是单选项的value，键值是单选框右边的文字描述。
checked 可选。选中值。如果是字符串或数组，表示选中的单选框。如果是数组，每个数组元素都是要选中的单选框值。
attributes 可选。同上。 
（8）checkbox(name, desc[, attributes, checked])




	

		名称
		描述
	

	

		name
		必需。同上。
	

		desc
		必需。复选框右边的文字。
	

	

		attributes
		可选。同上。
	

		

		checked
		可选。同上radio()。
	




（9）checkboxMulti(name, radios[, checked, attributes])
       生成批量复选框。复选框之间以空格分隔。




	

		名称
		描述
	

	

		name
		必需。同上。
	

		radios
		必需。单选框数组。键名是单选项的value，键值是单选框右边的文字描述。
	

	

		checked
		可选。同上。
	

		

		attributes
		可选。同上。
	




（10）select(name, options[, attributes, nullText]) 
      生成下拉框




	

		名称
		描述
	

	

		name
		必需。同上。
	

		options
		必需。option选项列表数组，键名选项值即value属性值，键值是显示文本
		  
	

	

		attributes
		可选。同上。不过如果是字符串或数字，表示默认选中的选项值。如果要设置select的value属性，则使用“{value}”作为属性数组中的键名。 
	

		

		nullText
		可选。下拉框一般有一个空选择的显示文本，如“请选择”，其值为空，表示未选中任何选项。
	




（11）selectGroup(name, options[, attributes, nullText])
      生成分组下拉框。




	

		名称
		描述
	

	

		name
		必需。同上。
	

		options
		必需。同上。每个选项组的第一个选项的选项值用数组，并且添加optgroup属性即可。
	

	

		attributes
		可选。同上。 
	

		

		nullText
		可选。同上。
	




（12）options(options[, value, nullText])
      仅生成option选项列表。如果不加select框，只生成option列表框，可以使用该方法。
      参数说明。




	

		名称
		描述
	

    

		options
		必需。同上。每个选项组的第一个选项的选项值用数组，并且添加optgroup属性即可。
	

	

		attributes
		可选。同上。 
	

		

		nullText
		可选。同上。
