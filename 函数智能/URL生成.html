URL生成函数



1、ocUrl(route[, params, relative, static, urlType])
       新建URL。
       特色：统一的URL生成函数，自动使用静态生成过的静态页面。
       参数说明：


	

		名称
		描述
	

	

		route
		必需。路由字符串或数组。
例如，admin模块的home控制器的index动作，这里可以为'/admin/home/index'或array('admin',
'home', 'index')。 
	

	

		params
		可选。参数数组。
	

	

		relative
		可选。表示是否生成相对路径。默认为false。如果设置为true，则URL不会加上协议和HOST，而以斜杠“/”开头，表示根目录。
	

	 

	  urlType
	  可选。不常用。URL路由类型。默认为系统配置的路由类型。
   

	

	  static
	  可选。不常用。是否检查静态页面是否存在。 默认为true，此时，如果静态页面存在，则返回静态页面的URL。
  



	实例如下：

假设当前域名是ocara.cn，如下：

//生成完整URL
$url = ocUrl('/item/index', array('id' => 2)); //返回http://www.ocara.cn/item/index/id/2
echo $url;

//生成相对URL
$url = ocUrl('/item/index', array('id' => 2)， true); //返回/item/index/id/2
echo $url;

2、ocRealUrl(dir[, subPath, root])
       类似于ocPath()，只不过是新建目录的URL。与ocUrl不同的是，ocUrl是MVC路由和参数的URL，而本函数是目录路径的URL。
       参数说明：


	

		名称
		描述
	

	

		dir
		必需。路径目录，参考ocPath()。 
	

	

		subPath
		可选。文件路径。
	

    	

		root
		可选。根URL。一般用于URL协议不是HTTP的情况。
	



       实例如下：

$ret = ocRealUrl('images', 'default/logo.jpg');
echo $ret; //输出类似于http://www.ocara.cn/public/images/default/logo.jpg

$ret = ocRealUrl('images', 'default/logo.jpg', 'ftp://www.ocara.cn');
echo $ret; //输出类似于ftp://www.ocara.cn/public/images/default/logo.jpg

3、ocSimpleUrl(dir[, subPath])
       生成相对URL。类似于ocRealUrl()，只不过是相对URL，前面去掉了协议和主机名。
       特色：统一的URL生成函数，自动使用静态生成过的静态页面。
       参数同上。
       实例如下：

$ret = ocSimpleUrl('images', 'default/logo.jpg');
echo $ret; //输出/public/images/default/logo.jpg
