Image

  

一、插件介绍

       图片插件，可以用来进行头像裁切、新建各种大小的缩略图、在任意位置加水印、水平反转或垂直反转。

特色：

智能处理。

二、方法函数介绍

1、setSrcImage(srcPath)
  设置源图片地址。  
  参数说明：
名称 描述
srcPath 必需。源图片地址

实例如下：

  		$imageFile = ocPath('attachments', 'default/red.jpg');
		$imageObj = new Image();
		$imageObj->setSrcImage($imageFile);

2、setDstImage(dstPath, mode)
  设置目标图片地址（要保存的）。  
  参数说明：
名称 描述
dstPath 必需。要保存的图片地址。
mode 必需。默认为0777。

实例如下：

$imageFile = ocPath('attachments', 'default/red2.jpg');
$imageObj = new Image();
$imageObj->setDstImage($imageFile);

3、clipImage(clipWidth, clipHeight, clipLx, clipLy, dstWidth, dstHeight, dstLx, dstLy)
  裁切图片。  
  参数说明：
名称 描述
clipWidth 必需。裁切宽度（决定了保存的背景宽度）
clipHeight 必需。裁切高度（决定了保存的背景高度）
clipLx 必需。裁切起点横坐标（决定了裁切起点的水平位置）
clipLy 必需。裁切起点纵坐标（决定了裁切起点的垂直位置）
dstWidth 可选。裁切之后保存宽度（决定新图本身的缩放宽度）
注：如果这个裁切宽度大于源图片本身的宽度，只会取源图片本身的宽度。
dstHeight 可选。裁切之后保存高度（决定新图本身的缩放高度）
注：如果这个裁切宽度大于源图片本身的高度，只会取源图片本身的高度。
dstLx 可选。裁切之后保存的起点横坐标（决定了新图本身相对于背景的水平偏移量）
dstLy 可选。裁切之后保存的起点纵坐标（决定了新图本身相对于背景的垂直偏移量）

实例如下：

1、图片裁切实例

         从源图片的(5, 10)处开始，裁切掉宽80像素和高85像素。


/*保存为30*35的新图*/
$imageObj = new Image();
$imageObj->setSrcImage(ocPath('attachments', 'default/red.jpg'));
$imageObj->setDstImage(ocPath('attachments', 'default/red_small.jpg'));

$imageObj->clipImage(80, 85, 5, 10, 30, 35);

/*保存为80*85的新图*/
$imageObj->setSrcImage(ocPath('attachments', 'default/red.jpg'));
$imageObj->setDstImage(ocPath('attachments', 'default/red_big.jpg'));

$imageObj->clipImage(80, 85, 5, 10);

2、位移裁切实例（用于头像裁切）
         位移，是指保存图片的起点有偏移，而不是从左上角的(0, 0)坐标，其实就是起点位置离左上角的有一定的距离。

/*
 * 从源图片的(5, 10)处开始，裁切掉宽80像素和高85像素
 * 保存为me_clip.gif，保存起点是(5,10)，保存宽度50像素，高度55像素
 * 注：如果裁切后的图像宽高不够50*55，比如是40*46，则保存也只会取实际的宽高40*46
 */
$imageObj = new Image();
$imageObj->setSrcImage(ocPath('attachments', 'default/me.gif'));
$imageObj->setDstImage(ocPath('attachments', 'default/me_clip.gif'));

$imageObj->clipImage(80, 85, 5, 10, 50, 55, 5, 10);

4、zoomImage(proportion, dstPath)
  新建缩略图。注:如果缩略图指定的宽度或高度大于图像本身的宽或高度，则只  
  参数说明：
名称 描述
proportion 必需。类似于以下： array( array('_small', 100, 200), //新建以_small结尾的缩略图 array('_big', 420, 500), //新建以_big结尾的大缩略图 )；
dstPath 必需。保存的图片路径。

实例如下：

假设test.gif图片的宽高分别是100和110像素。

$imageObj = new Image();
$imageObj->zoomImage(
	array(
		array('_small', 50, 50),
		array('_big', 60, 70),
     array('_over', 400, 500), //由于图片宽度不够，400和500都无效，只会取图片本身的宽100像素和高110像素
	), 
	ocPath('attachments', 'default/test.gif')
);

5、addMark(dstImage, markInfo, newFileSuffix)
  新建水印。  
  参数说明：
名称 描述
dstImage 必需。水印图片路径
markInfo 必需。水印参数。
newFileSuffix 可选。保存图片后缀。

实例如下：

文字

$imageObj->addMark($markImage, array(
	'text',
	array(
		  'location' => 'left-up',
		  'content' => 'A336',
		  'font' => 'Verdana',
		  'size' => '14',
		  'color' => '#FF0000'
	)
));

图片

$imageObj->addMark($markImage, array(
	'image', //图片类型
	array(
		'path' => ocPath('attachments', 'default/yaotou.gif'),  //水印图片
		'location' => 'left-up', //水平位置
		'transparent' => 5//透明度
	)
));

6、flipH(imageType)
  水平翻转图片。 可翻转通过setSrcImage()或setDstImage()设置的图片。
  参数说明：
名称 描述
imageType 可选。默认为src。可取src和dst两上值，含义如下：
（1）src：表示对源图片，即setSrcImage()设置的图片；
（2）dst：表示目标图片，即setDstImage()设置的图片。

实例如下：

假设test.gif图片的宽高分别是100和110像素。

		$imageObj = new Image();
		$imageObj->setSrcImage(ocPath('attachments', 'default/logo.gif'));
		$imageObj->setDstImage(ocPath('attachments', 'default/test.gif'));
		$imageObj->flipH('src'); //水平翻转logo.gif

7、flipH(imageType)
  垂直翻转图片。 也可翻转通过setSrcImage()或setDstImage()设置的图片。
  参数说明：
名称 描述
imageType 可选。同上。

实例如下：

假设test.gif图片的宽高分别是100和110像素。

		$imageObj = new Image();
		$imageObj->setSrcImage(ocPath('attachments', 'default/logo.gif'));
		$imageObj->setDstImage(ocPath('attachments', 'default/test.gif'));
		$imageObj->flipV('dst'); //垂直翻转test.gif
