Email

  

一、插件介绍

       邮件发送插件，可以方便地发送邮件。可以使用邮件服务器和Socket两种方法发送。

特色：

使用方便，没有自己的邮件服务器，也可以发送邮件，使用socket发送即可。

二、应用实例

（1）邮件配置

    /*
     * 邮箱配置
     */
    'EMAIL' => array(
        'sender'   => 'it123@163.com', //发送人，如myname@163.com
        'host'     => 'smtp.ym.163.com', //SMTP地址，如smtp.163.com
        'port'     => '25', //SMTP端口，如25
        'username' => 'it123', //登录用户名，如myname，有的可能是邮箱地址
        'password' => '123456', //登录密码，如my123456
    )


（2）邮件发送

$emailObj = ocService()->mail;

//设置收件人地址
$emailObj->setReceiver('123@qq.com');
//设置不添加回复内容 
$emailObj->noReply();  
//设置标题
$emailObj->setSubject('XX网邮箱验证（系统邮件，请勿回复）');
//设置html内容
$emailObj->setHtml($content); 
//socket发送
$emailObj->socketSend();



     注：以上使用的是邮箱配置。如果另外想重新设置邮箱配置信息，可以使用setServer()和setSender()等方法。


$emailObj->setServer('smtp.ym.163.com', 25, 'it123', '123456');
$emailObj->setSender('it123@163.com');

//发送邮件同上

三、方法函数介绍

1、setServer(host, port, username, password)
     设置SMTP服务器。  
     参数说明：
名称 描述
host 必需。服务器IP或域名。
port 必需。端口号。
username 必需。用户名。
password 必需。密码。

上面应用例子中，使用了邮箱配置。如果某处想重新设置邮箱服务器，则可以使用该方法设置，如：

 $emailObj->setServer('smtp.ym.163.com', 25, 'ocara', '123456'); 
  2、setSender(sender)
     设置发送人。  
     参数说明：
名称 描述
sender 必需。发送人邮箱地址

实例如下：

 $emailObj->sendSender('it@ocara.cn'); 
3、setReceiver(receiver[, cc, bcc])
     设置收件人。  
     参数说明：
名称 描述
receiver 必需。收件人邮箱地址
cc 可选。抄送人
bcc 可选。紧急抄送人 

4、setSubject(subject)
     设置标题。  
     参数说明：
名称 描述
subject 必需。标题 

5、localSend(header, params)
      本地服务器发送（需要将当前web服务器配置为邮件服务器）。  
      参数说明：
名称 描述
header 必需。设置邮件头
params 必需。参数 

6、socketSend()
      通过Socket发送邮件。如果没有自己的邮件服务器，一般可以通过这个来发送邮件。打开socket扩展即可。 
5、setHtml(html[, charset, encoding])
      设置HTML内容。
      参数说明：
名称 描述
html 必需。HTML内容
charset 可选。默认UTF-8。
encoding 可选。邮件传输编码。默认为quoted-printable，可打印字符引用编码。 
6、setText(text[, charset, encoding])
      设置文本内容。
      参数说明：
名称 描述
text 必需。文本内容
charset 可选。同上。
encoding 可选。同上。
