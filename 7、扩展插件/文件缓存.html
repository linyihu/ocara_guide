FileCache

  

一、插件介绍

       文件缓存插件，可以很方便地自动生成PHP文件缓存，还可以添加代码注释。

特色：

只需要指定要缓存的数组或字符串就可以了。

二、应用实例

假设，保存的数据为数组。

 $fileCache = new FileCache();

$users = array(
	array('小明', 12), 
	array('小刚', 13)
);

$filePath = ocPath('temp', 'users.php');

$fileCache->setData($users, 'users', '用户数据');
$fileCache->format();
$fileCache->save($filePath);

$content = $fileCache->read($filePath, 'users');
print_r($content);

执行结果：

（1）保存文件，内容为：

/**
 * 用户数据
 */
$users = array(
      array(
            '小明',
            12
      ),
      array(
            '小刚',
            13
      )
);


（2）打印输出：
Array ( [0] => Array ( [0] => 小明 [1] => 12 ) [1] => Array ( [0] => 小刚 [1] => 13 ) )

三、方法函数介绍

1、setData(data[, name, description])
  设置缓存数据。  
  参数说明：
名称 描述
data 必需。缓存数据，字符串或数组。
name 可选。要保存的变量名称。默认为空。如果不为空，则生成的缓存文件中，缓存内容会等于这个变量。否则，会以return返回。
decription 可选。代码注释。 
2、save(filePath[, append, perm])
  按动作action生成HTML。  
  参数说明：
名称 描述
filePath 必需。文件保存路径。
append 必需。是否追加到缓存文件末尾，而不是覆盖内容。
append 可选。如果文件不存在新建文件时添加的可读写权限，默认为0755。

实例如下：

$fileCache = new FileCache();

$users = array(
	array('小明', 12),
	array('小刚', 13)
);

$filePath = ocPath('data','test.php');
$fileCache->setData($users, 'users');
$fileCache->save($filePath);

3、read(filePath[, name])
  按动作action生成HTML。
  参数说明：
名称 描述
filePath 必需。文件路径。
name 可选。要获取的变量名称。 
$content = $fileCache->read(ocPath('temp', 'users.php'), 'users');
print_r($content);


3、format()
       格式化数据，如果数据是数组，保存的缓存数组，会按平时手写的数组格式保存。否则，会以var_export()生成。但是数组太大
时，没有性能没有后者快。
       比如上例中，如果在保存前，使用该函数后，可以看到保存的数组内容会有易读的格式。否则，如果是数字型数组，数字键名也显示出来了。
