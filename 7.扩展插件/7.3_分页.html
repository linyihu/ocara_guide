7.3 分页

  
一、插件介绍
       分页插件，对应类Ocara\Service\Pager。可以轻松地实现分页，每次切换分页后，都会显示当前页左右最近的页码，满足一般的分页需求。
二、公有属性介绍
       以下是公有属性，在页面模板中，可以自由使用。



	

		属性名称
		描述
	

		

		html
		必需。查询字符串中分页参数名称，一般默认为page。
	

		

		page
		当前页码。
	

		

		previosPage
		上一页页码。
	

			

		nextPage
		下一页页码。
	

			

		startPage
		开始页码。
	

			

		endPage
		结束页码。
	

				

		lastPage
		最后一页页码。
三、方法函数介绍 
1、setPage(page)
       设置当前页。
       参数说明：
名称 描述
page 必需。页码 
2、setPerPage(perPage)
       设置每页显示多少条记录。
       参数说明：
名称 描述
perPage 必需。每页显示的记录数 
3、setPageParam(pageParam)
       设置查询字符串中的分页参数的名称。
       参数说明：
名称 描述
pageParam 必需。参数名称。 
4、setPerShow(perShow)
       设置每次显示多少页。
       参数说明：
名称 描述
perShow 必需。每页显示的页数 
5、setUrl(url)
       设置分页URL。
       参数说明：
名称 描述
url 必需。页码的主要URL，可以是URL字符串或URL信息数组。其中不需要包含分页参数，分页参数会自动在该URL后面加上。 
6、setActiveClass(class)
       设置当前页CSS样式。
       参数说明：
名称 描述
class 必需。当页选中页的超链接的css class名称。 
7、setHtml(total, url[, attr])
       设置页码的主函数。
       参数说明：
名称 描述
total 必需。数据记录总数
url 必需。同上
attr 可选。参数数组。 
8、getInfo()
       返回分页信息。 返回分页信息，即Mysql的limit语句的两个参数：记录开始指针和查询数量

9、getPage()
       获取当前页。
