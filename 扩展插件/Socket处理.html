Socket

  

一、插件介绍

       Socket插件，实现了Socket的各种常用操作。

特色：

Socket服务端和客户端都写好了，使用方便了。

二、应用实例

1、服务端

 		error_reporting(E_ALL & ~E_NOTICE);
		$socket = new SocketServer();
		$conn = $socket->start('127.0.0.1', 1994);
		
		if ($conn) {
			$flag = 1;
			do {
				$socket->accept();
				$msg = "Send Message:Hello,World";
				if ($socket->send($msg)) {
					echo "T{$flag}:send Success\n";
				}
				
				if ($flag == 5) {
					echo "stop";
					break;
				}
				$flag++;
				$socket->close();
			}
			while (True);
			
		}

2、客户端

error_reporting(E_ALL & ~E_NOTICE);
		
$socket = new SocketClient();
$socket->connect('127.0.0.1', 1994);

$content = $socket->read(256));

三、方法函数介绍

（一）、服务端插件SocketServer

1、start(host, port[, timeout])
  该函数自动做了三步操作，分别是生成服务端、绑定端口和监听端口。  
  参数说明：
名称 描述
host 必需。客户端IP或域名。
port 必需。端口号。
timeout 可选。设置程序执行超时间，作为超时时间，其实使用的是 set_limit_time($timeout)函数。。 
2、accept()
  接收客户端请求。 
3、read(length[, type])
  读取指定长度的数据。  
  参数说明：
名称 描述
length 必需。要读取的字数，默认512个字符
type 必需。有PHP_BINARY_READ和PHP_NORMAL_READ。默认为PHP_BINARY_READ 
4、send(output[, length])
  发送数据。  
  参数说明：
名称 描述
output 必需。要发送的内容
length 必需。要发送的字符长度，默认512个字符 
5、close()
  关闭连接。

（二）、客户端插件SocketClient 

	6、connect(host, port[, limitTime, recTimeout, sendTimeout])
  接收客户端请求。  
  参数说明：
名称 描述
host 必需。同服务端的start()。
port 必需。同服务端的start()。
limitTime 可选。限制时间。
recTimeout 可选。接受超时时间。
sendTimeout 可选。发送超时时间。 
7、read(length[, type])
  读取指定长度的数据。  
  参数说明：
名称 描述
length 必需。同服务端
type 必需。同服务端 
8、send(output[, length])
  发送数据。  
  参数说明：
名称 描述
output 必需。同服务端
length 必需。同服务端
