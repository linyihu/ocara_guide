Date

  

一、插件介绍

       日期时间插件，可以对某日期时间进行各种操作。

特色：

任意地加减日期时间，参数用日期字符串、时间戳、或规定键值的数组都可以。

二、方法函数介绍

1、add(time, number, type[, format])
  增加时间参数。返回日期型字符串。  
  参数说明：
名称 描述
time 必需。日期时间参数。可以是字符串、时间戳或规定键名的数组（可以直接给出或通过getDateInfo()获取得到）
number 必需。增减数量。正数表示增加，负数表示减少。
type 必需。增加参数类型，包括年year、月month、日day、时hour、分minute和秒second。
format 可选。时间格式，'ymd'表示年月日，'mdy'表示月日年。默认为'ymd'年月日。

实例如下：

echo Date::add('2015-02-28 12:02:00', 3, 'month');
//输出2015-05-28 12:02:00 
2、set(time, number, type)
  设置时间参数。返回日期型字符串。  
  参数说明：
名称 描述
time 必需。同上
number 必需。同上
type 必需。同上

实例如下：

echo Date::set('2015-02-28 12:02:00', 3, 'month'); 
//输出2015-03-28 12:02:00 
3、get(time, type)
   获取时间参数。返回日期型字符串。  
  参数说明：
名称 描述
time 必需。同上
type 必需。同上

实例如下：

echo Date::get('2015-02-28 12:02:00', 'month'); 
//输出02 
4、getDate(time[, format])
  获取日期时间格式。返回日期型字符串。  
  参数说明：
名称 描述
time 必需。同上
format 可选。同上add()

实例如下：

//时间戳
echo Date::getDate(time(), 'mdy'); 
//输出当前月日年格式的时间字符串，比如：04-05-2013 13:13:24 

//规定键名的数组
$dateArray = Date::getDateInfo(time());
echo Date::getDate($dateArray, 'mdy'); 
//也输出月日年格式的当前时间字符串

//时间字符串
$dateStr = '2013-05-04 13:13:24';
echo Date::getDate($dateStr); 
//没有指定输出格式，默认输出年月日格式：2013-05-04 13:13:24

5、getInterval(startTime, endTime[, type])
获取时间间隔。  
  参数说明：
名称 描述
startTime 必需。开始时间，字符串或时间戳。
endTime 必需。结束时间，字符串或时间戳。
type 可选。获取类型，可以是days、hours、minutes或seconds。默认为空，会返回这四个情况的键值数组。

实例如下：

//时间字符串
$result = Date::getInterval('2015-02-28 12:15:13', '2015-03-05 13:25:13', 'hours');
echo $result; 
//输出1

//时间戳
$result = Date::getInterval('2015-02-28 12:15:13', time());
print_r($result); 
//没有指定type参数，输出类似于Array ( [days] => 36 [hours] => 1 [minutes] => 10 [seconds] => 2 ) 

//规定键名的数组
$dateArray = Date::getDateInfo('2015-02-28 12:15:13');
$result = Date::getInterval($dateArray, time());
print_r($result); 
//没有指定type参数，输出类似于上数组 
6、getDateInfo(time)
  获取日期信息。返回数组，日期格式不正确时，返回空数组。  
  参数说明：
名称 描述
time 必需。字符串或时间戳。注：不能是数组！

实例如下：

//时间字符串
ocPrint(Date::getDateInfo('2015-02-28 12:02:00'));
//输出Array ( [hour] => 12 [minute] => 02 [second] => 00 [month] => 02 [day] => 28 [year] => 2015 )

//时间戳
ocPrint(Date::getDateInfo(time()));
//输出类似

7、isYun(year)
  检测是否是闰年。返回true或false。  
  参数说明：
名称 描述
year 必需。数字型年份。

实例如下：

//1953非闰年
echo Date::isYun(1953); 
//输出false 
 
//2008是闰年
echo Date::isYun(2008);
//输出1
