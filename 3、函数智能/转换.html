转换函数



1、ocArray(data)
  　　对象递归转换成关联数组。
  　　参数说明：


	

		名称
		描述
	

	

		data
		对象
	





$obj = new stdClass();
$obj->a = 1;
$obj->b = 2;

$arr = ocArray($obj);

print_r($arr); //输出Array ( [a] => 1 [b] => 2 )

2、 ocObject(data)
      数组递归转换成对象。
      参数说明：


	

		名称
		描述
	

	

		data
		数组
	





$arr = array( 'a' => 1, 'b' => 2 );
$arr = ocObject($arr);

print_r($arr); //输出stdClass Object ( [a] => 1 [b] => 2 )
