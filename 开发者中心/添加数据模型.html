添加数据模型（Model）


（1）进入开发者中心，点击左边的"控制器（Model）"菜单。右边会出现新建控制器的界面。



添加选项介绍


数据库服务器名称
       必填，即application配置文件中配置的数据库服务器名，不是数据库名称。
       一般默认的是default，这对应于它的数据库连接配置:

/*
 * 数据库配置
 */
$CONF['DATABASE']['default'] = array(
	'type' 		=> 'mysql', //数据库类型
	'host' 		=> '', //数据库服务器主机和端口，默认端口可以省略
	'default' 	=> 'main', //默认数据库名称
	'username' 	=> '', //数据库用户名
	'password' 	=> '', //数据库用户密码
	'charset' 	=> '', //数据库编码，如果是utf8不要写成utf-8
	'prefix' 	=> '', //表名前缀
	'keywords'  => '', //SQL关键字
);


       $CONF['DATABASE']['default']的default即数据库服务器名。
数据库名称
       可选，数据库的名称。如果填写default或为空，则表示使用配置中的默认数据库名称，如上述配置代码中的main。
数据表名
       必填，数据表的名称。
主键字段
       必填，数据表的主键字段。
       可以是复合主键，各主键字段使用英文半角状态下的逗号","分隔，比如复合主键：detail_id,line
